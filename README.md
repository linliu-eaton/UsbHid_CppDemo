# UsbHid_CppDemo
An implementation of USBFW upgrade via USB HID, as demo for developing other HID functions.

# Build
## Build tool
Microsoft Visual Studio 2017 or later
## Note
Set solution platform to "Release" and "x86" for compiling, because the lib file(hid.lib, setupapi.lib) is for x86 and can't be used for x64.
